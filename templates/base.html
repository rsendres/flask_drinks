Trabalho 03 para Universidade Federal de Santa Catarina

O Trabalho 03 consiste no desenvolvimento de um pequeno programa Web em Python. O tema do trabalho é livre O programa deve ser desenvolvido utilizando o framework Flask Deve ser desenvolvida ao menos uma página dinâmica. Por página dinâmica, entende-se uma página Web que obtenha e exiba dados de um arquivo, banco de dados, ou outra fonte. Ou seja, as informações apresentadas na página não estão gravadas diretamente no html da página, mas são obtidos dinamicamente por meio de programação Python. Algumas sugestões: Reimplementar uma parte do Trabalho 2 para Web com uso do Flask Carregar dados obtidos de um arquivo ou banco de dados em uma página Web
